Quem Somos

Gamer World é uma loja virtual (e-commerce) fictícia criada como estudo de Desenvolvimento de uma Loja Virtual com angular 12 e Bootstrap 5

Foi utilizada uma api rest criada com a linguagem de programação java utilizando o framework Spring Boot e o banco de dados mysql para desenvolvimento e o banco de dados postgres para ambiente de produção a documentação foi feita utilizando a ferramentar swagger e o deploy utilizando a plataforma do heroku.
